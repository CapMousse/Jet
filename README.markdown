Jet
================

A lightweight, easy modifiable and fast PHP framework with MVC.
Tested on PHP 5.2+, PHP 5.3+

Features
--------

* PHP 5.3 support
* MVC support in OOP
* Modules support
* Personnal routes support
* Highly modifiable
* Light and fast
